网站接入QQ登录

admin 发表于 2013-10-23 9:42 Wednesday 超过634围观 0条评论

申请地址：http://connect.qq.com/

申请通过之后，得到APP ID和APP KEY，然后在 http://wiki.connect.qq.com/sdk%E4%B8%8B%E8%BD%BD#SDKfor.E7.BD.91.E7.AB.99.E6.8E.A5.E5.85.A5

下载相应语言的sdk,这里以php语言来介绍。

把下载的文件解压到网站目录下面，然后访问该目录，进入安装界面，把申请到的APP ID和APP KEY输入后安装成功。

进到sdk目录，找到exmaple里面的oauth文件，这个就是登录需要用到的，我们只需要修改callback文件，与自己的数据接入查询即可。

我给客户做的一个http://haoshuazuan.com淘宝刷销量的网站 http://haoshuazuan.com就使用了QQ登录下面是我的代码：

require_once("API/qqConnectAPI.php");

$qc = new QC();
$acs=$qc->qq_callback();  //获取 $acs
$openid=$qc->get_openid();   //获取 $openid
$qc = new QC($acs,$openid);  //根据 $acs,$openid 登录

$arr = $qc->get_user_info();  //获取用户资料

if(){   //如果有这个用户，则登录


}else{   //如果没有这个用户，则注册


}
////////跳回你要的网页
echo "<script>
window.location.href='index.php';
</script>";exit;
